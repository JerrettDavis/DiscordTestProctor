## What is Janus Proctor?
Janus Proctor is a Discord test proctoring bot and admin console. It lets administrators build certifications, run exams, and award roles with real-time visibility.

## Getting started
1. Invite the bot to your Discord server.
2. Sign in as an administrator.
3. Create or customize a certification template.
4. Add questions and publish.
5. Run `/get-certificates` in Discord to start exams.
